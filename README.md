# 🚀 TripX AI Planner

TripX AI Planner is a modern full-stack travel booking web application designed to provide an intelligent and immersive travel planning experience.

## 🎬 Features

- 🎥 Cinematic intro experience
- 👤 Personalized user welcome
- 🌍 Dynamic route visualization (any city input)
- 📏 Distance-based pricing system
- 🪑 Seat selection (Bus, Train, Flight)
- 🚆 Train coach & berth system (Superfast Express)
- 🚌 Premium bus services (VeeBee, KPN, etc.)
- ✈️ Flight availability check (airport-based logic)
- 🎟️ Multi-ticket booking support
- 💳 Simulated payment system (QR-based)
- 🧾 Smart ticket generation
- 🎁 Reward system (discount based on bookings)

## 💡 Pricing Logic

- 🚌 Bus → ₹1.7 per km  
- 🚆 Train → ₹1.25 per km  
- ✈️ Flight → Higher pricing based on availability  

## 🧠 Smart Features

- Dynamic city input (India-based)
- Real-time route generation
- Distance estimation (AI-simulated)
- Conditional flight availability

## 🛠️ Tech Stack

- **Frontend:** React + Vite  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Map:** Leaflet / Google Maps  
- **State Management:** React Hooks  

## 🎯 Highlights

- Unique scroll-to-modern map transition
- Real-world booking flow simulation
- Clean and premium UI/UX design
- Fully responsive and interactive
